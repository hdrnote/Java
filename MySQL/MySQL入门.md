# MySQL

SQL:Structured query language.

+------------------------------------------------------------------------------------------------------------------------------------------------------------+

### 开启/关闭MySQL服务

```
net start mysql80
net stop mysql80
```

### 打开MySQL

```
mysql -uroot -p
```

### 显示数据库

```mysql
show databases;
```

### 使用数据库

```mysql
use database_name;
```

### 显示表格

```mysql
show tables;
```

### 查看表格结构

```mysql
desc table_name;
```
