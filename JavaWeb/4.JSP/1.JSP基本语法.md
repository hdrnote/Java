# JSP基本语法

JSP可分为四种语法

1. 指令
2. 声明
3. 程序代码
4. 表达式

## 指令

用于设置和整个JSP页面相关的属性.

语法:

```jsp
<% @指令名 属性="值" %>
```

JSP中有三个常用指令page include taglib

## 声明

声明变量或方法

语法:

```jsp
<%! void sayHello{
    out.print("Hello World")
}%>
```

## 程序代码

语法

```jsp
<% out.print("Hello") %>
```

## 输出表达式

语法

```jsp
<%=表达式%>
```

## 提示:

你可能很好奇,为什么声明和程序代码要分开.

这是因为JSP脚本的内容全由_jspService()执行

方法中是不能定义方法的,所以需要用另一种语法来声明方法.
